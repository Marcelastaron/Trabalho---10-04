# 📘 Projetos de Estruturas de Dados em Java

Este repositório contém três projetos desenvolvidos em Java com o objetivo de demonstrar, na prática, o funcionamento de estruturas de dados e algoritmos fundamentais, além de comparar seu desempenho em diferentes cenários.

---

## 🌳 Projeto 1: Estruturas de Árvores

O primeiro projeto aborda a implementação de três tipos de árvores:

- **Árvore Binária de Busca (BST)**: permite inserção, remoção e busca de elementos de forma hierárquica. Seu desempenho pode ser prejudicado caso a árvore fique desbalanceada.
- **Árvore AVL**: uma versão balanceada da BST que utiliza rotações automáticas para manter a altura equilibrada, garantindo melhor eficiência nas operações.
- **Árvore Rubro-Negra**: também balanceada, utiliza regras de cores e rotações para manter a estrutura eficiente mesmo com muitas inserções.

Esse projeto demonstra como diferentes estratégias de balanceamento impactam diretamente no desempenho das operações.

---

## 🔍 Projeto 2: Algoritmos de Busca

O segundo projeto apresenta dois algoritmos principais de busca:

- **Busca Sequencial**: percorre todos os elementos até encontrar o valor desejado. É simples, porém menos eficiente em grandes volumes de dados.
- **Busca Binária**: divide o conjunto de dados ao meio a cada etapa, sendo mais eficiente, mas exigindo que os dados estejam ordenados.

Além disso, são realizados testes de desempenho comparando:

- Busca sequencial
- Busca binária
- Busca em estruturas de árvore

Os testes mostram, na prática, qual abordagem é mais eficiente dependendo do contexto.

---

## 🔄 Projeto 3: Algoritmos de Ordenação

O terceiro projeto foca em dois algoritmos clássicos de ordenação:

- **Bubble Sort**: realiza trocas sucessivas entre elementos adjacentes até que a lista esteja ordenada. É simples, mas pouco eficiente para grandes volumes de dados.
- **Insertion Sort**: insere cada elemento na posição correta dentro de uma lista já parcialmente ordenada, sendo mais eficiente em listas pequenas ou quase ordenadas.

Também são realizadas análises de desempenho considerando:

- Melhor caso
- Caso médio
- Pior caso

---

## 📊 Conclusão

Os três projetos se complementam ao demonstrar conceitos essenciais de estruturas de dados e algoritmos. Além da implementação, o trabalho destaca a importância da escolha adequada de cada estrutura ou algoritmo de acordo com o problema.

De forma geral:

- Árvores balanceadas garantem melhor desempenho em operações de busca e inserção.
- A ordenação é fundamental para a eficiência de alguns algoritmos, como a busca binária.
- A escolha do algoritmo de ordenação impacta diretamente no tempo de execução.
